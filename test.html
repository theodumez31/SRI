Développement

Réalisation et maintenance

PapIT est une entreprise spécialisée dans le développement informatique et la maintenance de backends d’applications dans les langages Python 3, CoBOL, Java et JavaScript. Nous mettons également à disposition notre expérience et maîtrise du langage de systèmes de gestion de base de données SQL(DB2, MySQL).

PapIT a collaboré avec des entreprises pour le développement de logiciels et d'applications populaires et très largement utilisées partout en France. Cela nous permet de garantir au mieux la maintenance d’applications et la gestion des données, ainsi que les challenges qui peuvent en découler.

Chez PapIT, nos développements sont autant que possible basés sur des technologies open source auditables. Ainsi, aucune licence payante n’est nécessaire pour leur exploitation. Des exemples de code sont disponibles sur la plateforme de développement GitHub.

Testing et qualité

Une des missions de PapIT consiste à réaliser le testing fonctionnel de tous types de logiciels : API web, réseau, embarqué. Pour ce faire, nous utilisons Pytest, Appium, Selenium, outils permettant de tester les IHM d’applications desktop, web, mobiles et embarquées de manière automatisée.

Egalement, nous utilisons le logiciel SonarQube dans le but de produire des KPI sur la qualité du code, permettant de guider le testing. Par la suite, les scripts de test développés et SonarQube sont intégrés dans des outils de supervision de qualité logicielle, comme le logiciel de CI/CD Jenkins. Cela permet de faciliter le contrôle qualité et la livraison du produit final.
